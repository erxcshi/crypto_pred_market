## Python

Use Python `3.13.0` for local development so it matches the repo virtualenv and notebook kernel metadata.

Quick setup:

```bash
python3.13 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
